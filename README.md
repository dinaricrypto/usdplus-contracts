## Development

### Prerequisites

For coverage report generation:
```shell
brew install lcov
```

### Build

```shell
$ forge build
```

### Test

```shell
$ yarn test
```

### Format

```shell
$ forge fmt
```

### Gas Snapshots

```shell
$ forge snapshot
```
